## Define global variables

Setting a global variable saves global search and replaces later. Declare global variables in the `/src/data/Variables.yml` file using the following syntax:

```yaml
<variable-name>: <variable-value>
```

**To use a global variable in a topic:**

1. Declare or choose a variable in the `Variables.yml` file.

   > /src/data/Variables.yml

   ```yaml
   version: 1.0
   exampleVariable: This is an example variable
   hackTeam: Project Hyde
   ```

1. Use the variable inline within your topic content. This example contains two variables.

   ```
   This is version <Variable name='version'/> of a Doc site created by <Variable name='hackTeam'/> Hackathon team!
   ```

   Sample response:

   This is version <Variable name='version'/> of a Doc site created by <Variable name='hackTeam'/> Hackathon team!
