# Configure menus

The site navigation includes various menu types that you can customize with a YAML file.

## Section menus

Section menus are in the `src/data/pageGroups/` directory. The top menu, or tabs, organize the site into sections. Each section presents a unique menu of topics on the left side.

To add a new section to the tab menu, create a YAML file in the `src/data/pageGroups/` directory and:

```yaml
order: 1
title: About
name: section-1
url: /
pages:
  - title: Home
    url: /
```

- _order_ controls the tab position in the top menu.
- _title_ is the visible tab name.
- _name_ is the section name, such as `cloud-guide`.
- _url_ is the relative path of the markdown page to introduce the section.

The example returns the _About_ tab in the first position and the _Home_ page as the first topic in the left-side menu:

## Footer menu

The footer menu is a basic menu at the bottom of the site that includes the legal and copyright information. Update the footer menu in the `/data/footer.yml` file.

![footer][]

## Topic menu

The topic menu, titled **On This Page**, is on the right-hand side of the topic view. This menu automatically provides a list of the current page contents based on the header level. This TOC does not recognize header size greater than H3.

To update the title of this TOC, change the **spectrum-SideNav-heading** in the `src/components/TableOfContents/tableOfContents.js` file.

## Waffle menu

The waffle menu is useful for providing links to external documentation. To customize the waffle menu, use the `/data/externalPages/externalpages.yml` file.
